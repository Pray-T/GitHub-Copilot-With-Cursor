GitHub URL 입력 안내

아래는 자주 하는 실수 예시입니다.

  • 맨 뒤 / 제거
    잘못된 예 https://github.com/owner/repo/
    올바른 예 https://github.com/owner/repo
  • https 사용 (http 불가)
    잘못된 예 http://github.com/owner/repo
    올바른 예 https://github.com/owner/repo
  • 브랜치·파일 경로 제거 (주소창 복사 시 흔함)
    잘못된 예 https://github.com/owner/repo/tree/main
    올바른 예 https://github.com/owner/repo
  • 올바른 형식
    형식 https://github.com/{owner}/{repo}

예시의 상황이 아닌지 확인해주세요.
